Babylon AR

The following list provides links to the available Babylon resources and capabilities. Please keep in mind that Babylon AR is in very early pre-release, and the links and offerings are subject to change.

  • babylonAr.js: The full currently-available API.
  • babylonAr.playground.js: A version of babylonAr.js a hack that allows it to be loaded in the Babylon Playground using BABYLON.Tools.LoadScript().
  • exampleWorker.js: Example of creating a WASM-back WebWorker.
  • exampleObjectTracker.js: Example of basic object tracker.
